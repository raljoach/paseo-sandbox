# Paseo Airbnb Analyzer

Paseo is a tool for collecting Airbnb listings, analyzing value, capturing user preferences, training a recommendation model, and generating predictions.

## Requirements

- Python 3.12+
- Node.js
- npm

## Python Setup

pip install -r requirements.txt
npm install
cd paseo        
python -m dashboard.app

The data/ directory is generated and intentionally ignored by git.

It contains:

scraped Airbnb listings
processed features
likes
model predictions
